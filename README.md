# Pakistani Politician Classifier 🇵🇰

This project is an AI-based image classification system that identifies Pakistani politicians using Deep Learning and Transfer Learning techniques.

## Features
- Image upload prediction system
- ResNet50 CNN model
- Streamlit frontend
- 16 politician classes
- Transfer learning and fine-tuning

## Technologies Used
- Python
- TensorFlow
- Keras
- Streamlit
- NumPy
- Pillow

## Best Model
ResNet50 achieved approximately 89% test accuracy.

```bash
python -m streamlit run app.py
